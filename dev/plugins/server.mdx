Server-side plugins are written in Go and run as separate processes that communicate with the Mattermost server via RPC. They can implement hooks to respond to events and use the Plugin API to interact with the server.

## Plugin Structure

Every server plugin must:

1. Embed `plugin.MattermostPlugin`
2. Implement desired hooks from the `Hooks` interface
3. Call `plugin.ClientMain()` in the `main()` function

### Basic Structure

```go
package main

import (
    "github.com/mattermost/mattermost/server/public/model"
    "github.com/mattermost/mattermost/server/public/plugin"
)

type Plugin struct {
    plugin.MattermostPlugin
}

func (p *Plugin) OnActivate() error {
    p.API.LogInfo("Plugin activated")
    return nil
}

func main() {
    plugin.ClientMain(&Plugin{})
}
```

## Plugin API

The Plugin API is available via `p.API` and provides methods to interact with Mattermost.

### Accessing the API

```go
func (p *Plugin) MessageHasBeenPosted(c *plugin.Context, post *model.Post) {
    // API is available as p.API
    user, err := p.API.GetUser(post.UserId)
    if err != nil {
        p.API.LogError("Failed to get user", "error", err.Error())
        return
    }
    
    p.API.LogInfo("User posted", "username", user.Username)
}
```

### Configuration and Logging

```go
// Get server configuration
config := p.API.GetConfig()

// Get plugin configuration
pluginConfig := p.API.GetPluginConfig()

// Load plugin settings into struct
var config MyPluginConfig
err := p.API.LoadPluginConfiguration(&config)

// Logging
p.API.LogDebug("Debug message", "key", value)
p.API.LogInfo("Info message", "key", value)
p.API.LogWarn("Warning message", "key", value)
p.API.LogError("Error message", "error", err.Error())

// Get server version
version := p.API.GetServerVersion()

// Check license
license := p.API.GetLicense()
isEnterprise := p.API.IsEnterpriseReady()
```

## Hooks

Hooks are methods you implement to respond to events in Mattermost.

### Lifecycle Hooks

<ParamField path="OnActivate" type="func() error">
  Called when the plugin is activated. Use this to:
  - Register slash commands
  - Initialize resources
  - Start background jobs
  - Validate configuration

  Return an error to prevent activation.

  ```go
  func (p *Plugin) OnActivate() error {
      if err := p.API.RegisterCommand(&model.Command{
          Trigger: "myplugin",
      }); err != nil {
          return fmt.Errorf("failed to register command: %w", err)
      }
      
      p.API.LogInfo("Plugin activated successfully")
      return nil
  }
  ```
</ParamField>

<ParamField path="OnDeactivate" type="func() error">
  Called when the plugin is deactivated. Clean up:
  - Stop background workers
  - Close connections
  - Release resources

  ```go
  func (p *Plugin) OnDeactivate() error {
      p.stopBackgroundJobs()
      p.API.LogInfo("Plugin deactivated")
      return nil
  }
  ```
</ParamField>

<ParamField path="OnConfigurationChange" type="func() error">
  Called when plugin configuration changes. Load new settings:

  ```go
  func (p *Plugin) OnConfigurationChange() error {
      var config MyPluginConfig
      if err := p.API.LoadPluginConfiguration(&config); err != nil {
          return err
      }
      
      p.setConfiguration(&config)
      return nil
  }
  ```
</ParamField>

### Message Hooks

<ParamField path="MessageWillBePosted" type="func(*plugin.Context, *model.Post) (*model.Post, string)">
  Called before a message is saved. You can:
  - Modify the message
  - Reject the message
  - Allow it unchanged

  **Return values:**
  - `(modifiedPost, "")` - Allow with modifications
  - `(nil, "")` - Allow unchanged
  - `(nil, "rejection reason")` - Reject with reason
  - `(nil, plugin.DismissPostError)` - Silently dismiss

  ```go
  func (p *Plugin) MessageWillBePosted(c *plugin.Context, post *model.Post) (*model.Post, string) {
      // Block messages with profanity
      if containsProfanity(post.Message) {
          return nil, "Message contains inappropriate content"
      }
      
      // Add hashtag to messages
      post.Message = post.Message + " #fromplugin"
      return post, ""
  }
  ```
</ParamField>

<ParamField path="MessageHasBeenPosted" type="func(*plugin.Context, *model.Post)">
  Called after a message is saved. Use for:
  - Triggering actions based on content
  - Sending notifications
  - Logging or analytics

  ```go
  func (p *Plugin) MessageHasBeenPosted(c *plugin.Context, post *model.Post) {
      if strings.Contains(post.Message, "@bot") {
          p.API.CreatePost(&model.Post{
              ChannelId: post.ChannelId,
              Message:   "You mentioned me!",
          })
      }
  }
  ```
</ParamField>

<ParamField path="MessageWillBeUpdated" type="func(*plugin.Context, *model.Post, *model.Post) (*model.Post, string)">
  Called before a message is updated.

  ```go
  func (p *Plugin) MessageWillBeUpdated(c *plugin.Context, newPost, oldPost *model.Post) (*model.Post, string) {
      // Prevent removing mentions
      if containsMentions(oldPost.Message) && !containsMentions(newPost.Message) {
          return nil, "Cannot remove mentions"
      }
      return newPost, ""
  }
  ```
</ParamField>

### HTTP Hook

<ParamField path="ServeHTTP" type="func(*plugin.Context, http.ResponseWriter, *http.Request)">
  Handle HTTP requests to `/plugins/{plugin-id}/*`

  ```go
  func (p *Plugin) ServeHTTP(c *plugin.Context, w http.ResponseWriter, r *http.Request) {
      switch r.URL.Path {
      case "/api/status":
          p.handleStatus(w, r)
      case "/api/webhook":
          p.handleWebhook(w, r)
      default:
          http.NotFound(w, r)
      }
  }

  func (p *Plugin) handleStatus(w http.ResponseWriter, r *http.Request) {
      w.Header().Set("Content-Type", "application/json")
      w.WriteHeader(http.StatusOK)
      json.NewEncoder(w).Encode(map[string]string{
          "status": "ok",
      })
  }
  ```

  The `Mattermost-User-Id` header contains the authenticated user ID (if any).
</ParamField>

### Command Hook

<ParamField path="ExecuteCommand" type="func(*plugin.Context, *model.CommandArgs) (*model.CommandResponse, *model.AppError)">
  Handle slash command execution.

  ```go
  func (p *Plugin) ExecuteCommand(c *plugin.Context, args *model.CommandArgs) (*model.CommandResponse, *model.AppError) {
      switch args.Command {
      case "/myplugin help":
          return &model.CommandResponse{
              Text: "Available commands: /myplugin status, /myplugin config",
          }, nil
          
      case "/myplugin status":
          return &model.CommandResponse{
              ResponseType: model.CommandResponseTypeEphemeral,
              Text:         "Plugin is running",
          }, nil
          
      default:
          return &model.CommandResponse{
              Text: "Unknown command",
          }, nil
      }
  }
  ```
</ParamField>

## Slash Commands

Register custom slash commands users can invoke:

```go
func (p *Plugin) OnActivate() error {
    return p.API.RegisterCommand(&model.Command{
        Trigger:          "myplugin",
        DisplayName:      "My Plugin Command",
        Description:      "Execute my plugin command",
        AutoComplete:     true,
        AutoCompleteDesc: "Available commands: status, config",
        AutoCompleteHint: "[status|config]",
    })
}
```

## Publishing WebSocket Events

Send real-time events to connected clients:

```go
p.API.PublishWebSocketEvent("custom_event", map[string]interface{}{
    "message": "Hello from plugin",
    "data":    someData,
}, &model.WebsocketBroadcast{
    UserId:    userID,     // Send to specific user
    ChannelId: channelID,  // Or send to channel
})
```
