# Welcome to Mattermost

Mattermost is an **open core, self-hosted collaboration platform** that offers chat, workflow automation, voice calling, screen sharing, and AI integration. Built for teams that need security, control, and customization, Mattermost runs as a single Linux binary and relies on PostgreSQL for data storage.

## Why Mattermost?

Mattermost combines the familiarity of modern messaging with enterprise-grade security and compliance. Written in **Go and React**, it delivers high performance while maintaining complete data ownership.

## Key Technical Details

- **Backend**: Go 1.24.13
- **Frontend**: React (version 11.4.0)
- **Database**: PostgreSQL 14+
- **Deployment**: Single Linux binary, Docker, or Kubernetes
- **License**: MIT license (released monthly on the 16th)
- **Architecture**: Designed for high availability with clustering support

## Security & Updates

<Warning>
  Mattermost follows a **mandatory upgrade policy** with updates provided for the latest 3 releases and the current Extended Support Release (ESR). Subscribe to the [Security Bulletins Mailing List](https://mattermost.com/security-updates/#sign-up) for critical security updates.
</Warning>

Security issues should be reported to [responsibledisclosure@mattermost.com](mailto:responsibledisclosure@mattermost.com). Learn more in our [Security Policy](https://mattermost.com/security-vulnerability-report/).
